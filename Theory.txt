5. 외부 함수 isValid는 Student 클래스의 private 접근 권한을 가진 변수 id에 접근할 수 없음.
class Student;
bool isValid(Student s);
class Student {
	int id;
public:
	Student(int id) { this->id = id; }
	friend bool isValid(Student s);
};
6. 외부 함수 show는 Student 클래스의 변수 id 및 Professor 클래스의 변수 name에 접근할 수 없음.
class Student;
class Professor;
class Student {
	int id;
public:
	Student(int id) { this->id = id; }
	friend void show(Student s, Professor p);
};
class Professor {
	string name;
public:
	Professor(string name) { this->name = name; }
	friend void show(Student s, Professor p);
};
